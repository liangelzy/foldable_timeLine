# foldable_timeLine
这是一个可以折叠的时间轴

目前有点小问题，关闭动画好像没有

## 功能：
> 内容可以按照时间线，展示有标题，内容，时间，作者等，可以自由扩展，标题可以自定义点击事件，折叠时动画效果没弄出来
